# Privacy Policy for Naviance Viewer

Last updated: May 19, 2026

Naviance Viewer helps users view, save, compare, and export Naviance scattergram data that they can already access in their own browser.

## Data the Extension Handles

When enabled on supported Naviance pages, Naviance Viewer may read Naviance scattergram page content and scattergram API responses. This data may include college names, anonymized applicant GPA and test score ranges, application outcomes, application year summaries, and related Naviance scattergram fields. The extension does not save Naviance user profile fields such as user IDs or account information.

The extension also stores user preferences, such as whether to automatically parse supported Naviance pages and whether capture mode is enabled.

## How Data Is Used

Naviance Viewer uses this data only to provide its single purpose: parsing Naviance scattergram data, showing summaries, saving local school records, comparing colleges, and exporting the user's saved data as JSON or CSV.

## Storage

Parsed scattergram records are stored locally in the browser using extension IndexedDB. Extension preferences are stored using Chrome extension storage. Naviance Viewer does not upload scattergram records to a developer server.

## Sharing and Sale of Data

Naviance Viewer does not sell user data. Naviance Viewer does not transfer user data to third parties, except when the user explicitly exports a JSON or CSV file and chooses what to do with that file.

## Remote Code and External Services

Naviance Viewer does not load or execute JavaScript or WebAssembly from remote servers. The extension's executable code is included in the extension package.

## Limited Use

Naviance Viewer's use of user data is limited to providing and improving its single purpose. Naviance Viewer does not use or transfer user data for personalized advertising, creditworthiness, lending, or purposes unrelated to the extension's single purpose.

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

## Contact

For privacy questions, contact the publisher at the contact email listed on the Chrome Web Store listing.
